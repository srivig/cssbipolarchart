CSS Bipolar Chart

Just a simple jQuery plugin to draw html5/CSS bipolar bar chart and it is fairly responsive :)

For bipolar chart

$("#example1").drawCSSBipolarChart({
          data: sampleData1,
          bipolar: true
})

Data Format

[
  ["left label 1", "right label 1", 0.3],
  ["left label 2", "right label 2", 0.4]
]
